Beschrijving:

Spiralen en hoekige figuren in verschillende kleuren met python turtle.

Code:

import turtle

# Functie om RGB-kleuren correct in te stellen
def rgb_color(r, g, b):
    return (r/255, g/255, b/255)

t = turtle.Turtle()
t.speed(0)  # 0 betekent snelste snelheid
t.pensize(10)

t.color(rgb_color(0, 255, 205))

for i in range(360):
    t.circle(i, 10)

t.penup()
t.home()
t.pendown()
t.color("white")
t.pensize(12)

# Teken een cirkelpatroon

    my_turtle.pencolor(colors[_ % 4])
